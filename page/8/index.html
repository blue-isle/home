RSSHub 部署笔记

2020-03-21 00:00:00 #折腾 

🍰 万物皆可 RSS。

内容来源：https://docs.rsshub.app/install/#an-zhuang-2

$ docker pull diygod/rsshub
$ docker run -d --name rsshub -p 1200:1200 diygod/rsshub

用宝塔面板的话，放行 1200 端口。浏览器打开 http://IP:1200/ ，看到成功页面。✅
